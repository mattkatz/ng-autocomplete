WHY DOES NOTHING SHOW
Autocompleting in angular should be pretty easy, no? I wrote this so that I could have an easy way to mimic the tagging function in WordPress in Orbital Feed Reader.

The simplest thing is to give a data-suggestion-source of an array and a model to store the value in.

Here the array is: ["{{item}}",]

You can change the multiple-separator from the default of ", " to something like "|"

By default, the min-length before suggestions start appearing is 2 characters. Here min-length is 4. Try typing in "poli".

You can specify a specific css class for the selected item. Here we'll change from the default of 'selected' to blueselected
