Angular Autocomplete

I'm creating this directive because I want to do tagging on Orbital Feed Reader and it seems like a tagging input is a clear option for a directive. It should behave pretty similar to how suggest.js works in WordPress.

How it works

The simplest thing is to give a data-suggestion-source of an array and a model to store the value in.

Here the array is:

tagSource = ["{{item}}",];

<mk-autocomplete data-ng-model="tags" data-suggestion-source="tagSource" multiple="true" data-multiple-separator="," data-min-length="2" data-select-class="selected"/>

Try typing 'po' or 'lo' into this example. Use the arrow keys to go up or down, use enter or tab to select an option

Required

ng-model

Stores accepted suggestions in ng-model. Right now it's just connected as a string, you have to parse things into an array if that's what you want.

Now your ng-model contains {{tags}}

suggestionSource

accepts an array.

Not yet done: pass in a function that accepts 0 or 1 parameters. The function will be passed the current fragment

Optional

multiple

Should we accept multiples? Default is true.

multipleSeparator

You can change the multiple-separator from the default of ", " to something like "|"

<mk-autocomplete id='autosuggest' data-suggestion-source='tagSource' data-ng-model='tags' data-multiple-separator='|'></mk-autocomplete>

minLength

By default, the min-length before suggestions start appearing is 2 characters. Here min-length is 4. Try typing in "poli".

<mk-autocomplete id='autosuggest' data-suggestion-source='tagSource' data-ng-model='tags' data-min-length='4'></mk-autocomplete>

selectClass

You can specify a specific css class for the selected item. Here we'll change from the default of 'selected' to 'blueselected'

<mk-autocomplete id='autosuggest' data-suggestion-source='tagSource' data-ng-model='tags' data-select-class='blueselected'></mk-autocomplete>

onAccept - not working yet, still todo

triggers when user does any of the accept actions: tab, enters multiple sep, hits down/up arrows and then enter.
